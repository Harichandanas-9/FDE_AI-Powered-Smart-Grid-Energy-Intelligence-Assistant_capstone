Core Technology Choices

🤖 LLM — Groq

llama-3.3-70b-versatile. 200+ tokens/sec. Free tier. 10s timeout, 0 retries. Fallback chain: Groq → OpenAI → Template (offline).

langchain-groq

🔍 Search — BM25

Keyword index. Works offline. No GPU. No API key. TF-IDF weighted — perfect for technical grid domain vocabulary.

rank-bm25

🗄️ Vector DB — ChromaDB

Semantic search (disabled on Windows/Python 3.13 due to crash). BM25 is primary. ChromaDB activates on Linux production.

chromadb 1.5.9

🔀 LangGraph

StateGraph + MemorySaver for /chat/query endpoint. Sequential 6-node pipeline. Session memory across turns. NOT SqliteSaver (broke in 0.2.50+).

langgraph

🔗 LangChain

Unified LLM abstraction. ChatGroq, ChatOpenAI, ChatAnthropic all use same .invoke() interface. Swap provider by changing one config line.

langchain-core

⚡ FastAPI

Async ASGI server. Auto Swagger docs at /docs. Pydantic validation. 17 endpoints. uvicorn --loop asyncio (required for Python 3.13 Windows).

fastapi · uvicorn
